# Jupyter Notebooks

Interactive tutorials and experiments for the XAI-SHAP Framework.

## Available Notebooks

### 01_quick_start.ipynb

Quick introduction to the framework covering:

- Data loading
- Model training
- SHAP explanations
- Basic visualizations

### 02_shap_deep_dive.ipynb

In-depth exploration of SHAP values:

- Types of SHAP explainers
- Local vs global explanations
- Dependence plots
- Interaction values

## Running Notebooks

```bash
# Start Jupyter Lab
jupyter lab

# Or Jupyter Notebook
jupyter notebook
```

## Requirements

Make sure you have Jupyter installed:

```bash
pip install jupyter jupyterlab
```

Or use VS Code with the Jupyter extension.
